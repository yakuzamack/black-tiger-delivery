## 📋 Quick Setup for GitHub Pages

### 1. Create GitHub Repository
```bash
# Create a new repository on GitHub named "black-tiger-delivery"
# Then connect your local project:

git init
git add .
git commit -m "Initial commit: Black Tiger Delivery website"
git branch -M main
git remote add origin https://github.com/USERNAME/black-tiger-delivery.git
git push -u origin main
```

### 2. Enable GitHub Pages
1. Go to your repository on GitHub
2. Navigate to **Settings** → **Pages**
3. Under **Source**, select **GitHub Actions**
4. The website will auto-deploy on every push to `main` branch

### 3. Update Configuration
Replace `USERNAME` in these files with your GitHub username:
- `package.json` (homepage & repository URLs)
- `vite.config.ts` (base path)
- `README.md` (demo link)

## 🛠️ Local Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🌐 Deployment Options

### Automatic Deployment (Recommended)
- **GitHub Actions**: Deploys automatically on push to `main`
- **URL**: `https://USERNAME.github.io/black-tiger-delivery/`

### Manual Deployment
```bash
# Install gh-pages globally (optional)
npm install -g gh-pages

# Deploy manually
npm run deploy
```
